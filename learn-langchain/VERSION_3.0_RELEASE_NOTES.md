## ✨ 主要变化

### 1. 📁 文件结构重组

**之前**:
```
learn-langchain/
├── examples/         # 所有示例混在一起
├── docs/            # 文档
└── learn/           # 各种学习材料混在一起
```

**现在**:
```
learn-langchain/
├── week1_langchain_basics/      # Week 1: 按周组织
├── week2_agent_tools/           # Week 2: 清晰的学习路径
├── week3_langgraph_basics/      # Week 3: 每周有独立目录
├── ...                          # ...
├── source_code_guide/           # 源码学习指南
├── docs/                        # 辅助文档
└── archive/                     # 旧内容归档
```

**优势**:
- ✅ 学习路径清晰
- ✅ 内容按难度组织
- ✅ 每周独立完整
- ✅ 易于维护和更新

## 📊 迁移指南

### 如果你正在使用旧版课程

#### 方案 1: 继续完成旧版（不推荐）
旧版内容在 `archive/` 目录中，仍可使用，但不再更新。

#### 方案 2: 迁移到新版（推荐）

**映射关系**:
- 旧版 Day 1-7 → 新版 Week 1
- 旧版 Day 8-14 → 新版 Week 2
- 旧版 Day 15-28 → 新版 Week 3-4
- 旧版 Day 29-42 → 新版 Week 5-6
- 旧版 Day 43-60 → 新版 Week 7-8

**迁移步骤**:
1. 查看新版 `CURRICULUM_V3.md` 了解整体结构
2. 找到你当前学习进度对应的新版Week
3. 从新版对应的Week开始学习
4. 使用新版的代码和项目

### 如果你是新学员

直接从新版开始学习：
```bash
# 1. 阅读快速导航
cat QUICK_START_GUIDE.md

# 2. 查看完整大纲
cat CURRICULUM_V3.md

# 3. 开始 Week 1
cd week1_langchain_basics
python day1_hello_langchain.py
```
